# Why "Angles"?  
Well, I incorrectly translated word "Уголки" to English. I believe correct 
name of this game is "Corners".

# Prerequisites
 * GCC with C++20 support
 * Cmake >=3.12
 * GLFW 3.x
 * GLEW

# Building
Manually:
```
$ mkdir cmake_build
$ cd cmake_build
$ cmake ..
$ make
$ cd ..
```

With shortcut:
```
$ make build_release
```

# Controls
Left mouse button - choose boy  
Right mouse button - move boy  
Esc - Quit game
